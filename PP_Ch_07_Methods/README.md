## Calling Methods the Object-Oriented Way



```python 
>>> 'browning'.capitalize()
'Browning'
>>> 'Sonnet 43'.center(26)
'        Sonnet 43         '
>>> 'How do I love thee?  Let me count the ways.'.count('the')
2
``` 

## Exploring String Methods




```python

False

``` 

```python 
>>> 'species'.endswith('a')

``` 


```python 
>>> compound = '     \n  Methyl \n butanol   \n'
>>> compound.lstrip()
'Methyl \n butanol   \n'
>>> compound.rstrip()
'     \n  Methyl \n butanol'
>>> compound.strip()
'Methyl \n butanol'

```


## What Are Those Underscores?



```python 
>>> 'TTA' + 'GGG'
'TTAGGG'
>>> 'TTA'.__add__('GGG')
'TTAGGG'

``` 

```python 
Help on class int in module builtins:

class int(object)
...
 |  Methods defined here:
 |
 |  __abs__(self, /)
 |      abs(self)
 |
 |  __add__(self, value, /)
 |      Return self+value.
 |
 |  __gt__(self, value, /)

``` 



```python 
>>> abs(-3)
3
>>> (-3).__abs__()
3
>>> -3 .__abs__()

>>> -(3 .__abs__())

>>> (-3) .__abs__()
3
>>> 3 + 5

>>> 3 .__add__(5)

>>> 3 > 5

>>> 3 .__gt__(5)
``` 
